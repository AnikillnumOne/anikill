Урок 2. Основы HTML

Итак, давайте начнём знакомиться с синтаксисом HTML.

Любая веб-страница состоит из элементов, это может быть текст, кнопка, поле ввода и т. п.

Теги

Элемент состоит из открывающего тега, закрывающего тега и контента между ними.

Чтобы создать открывающий тег, нужно написать имя тега и обернуть его в угловые скобки. Например, код, чтобы создать открывающий тег кнопки:

<button>

А чтобы создать закрывающий тег, нужно так же обернуть имя тега в угловые скобки, но поставить слеш:

</button>

Контентом для кнопки служит текст, который будет на ней.

Создадим кнопку с текстом "нажми": <button>Нажми</button>

У нас появится серая кнопка с соответствующим текстом. При нажатии пока что ничего не происходит.

Существует огромное множество различных тегов. Знать их все совсем не нужно.

Рассмотрим, например, тег для текста:

<p>Это просто текст</p>

Мы рассмотрим наиболее популярные и важные теги.
