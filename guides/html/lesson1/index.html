Урок 1. Вступление и настройка среды разработки

Что такое HTML?

HTML (hypertext markup language, язык гипертекстовой разметки) — язык разметки для создания и структурирования веб-страниц. Он не является языком программирования и довольно прост в освоении.

Как писать на HTML?

Браузеры сами читают и отрисовывают элементы HTML, поэтому чтобы начать писать на этом языке, Вам достаточно иметь всего лишь 1 инструмент — текстовый редактор.

Давайте попробуем написать наш первый сайт в блокноте. Открываем соответствующую программу и вводим следующий код (разберём его позже):

<!doctype html>
<html>
    <body>
        Привет, мир!
    </body>
</html>

Сохраняем файл с расширением .html, копируем путь к нему и вставляем это в адресную строку браузера: у нас откроется веб-страница с текстом "Привет, мир!". Она доступна только Вам, так как является локальной и хранится в памяти компьютера. Если Вы внесли изменения в код, нажмите F5 (обновить страницу). Иногда может потребоваться обновление кэша: Ctrl + F5.

Установка и настройка Visual Studio Code

Можно писать в блокноте, Notepad++ или на сайте Github.com, но я рекомендую использовать самый популярный редактор — Visual Studio Code.

Visual Studio Code - интегрированная среда разработки (IDE) для работы с различными языками программирования, фреймворками, инструментами и т. д.

Переходим на сайт Microsoft и скачиваем данную среду разработки и запускаем установщик. После установки закройте приветствующую вкладку, нажмите File - Create a new file. Мы создаём новый файл. Назовём его index.html (можете выбрать другое имя, но обязательно используйте расширение .html). Здесь мы будем писать код. Рекомендую подробнее ознакомиться с интерфейсом VS Code: мы можем открывать уже существующие файлы и папки и управлять ими.

Также данная IDE содержит множество расширений, они добавляют разнообразные инструменты, повышают комфорт во время разработки. Рекомендую скачать Russian Language Pack for Visual Studio Code и Material Theme. Первая из них устанавливает русский язык для интерфейса, а вторая добавляет красивую подсветку синтаксиса и улучшает дизайн самой IDE (можете выбрать другие расширения для темы). Можете поискать расширения HTML для подсвечивания ошибок.

Для установки расширения нажмите на левой панели соответствующую кнопку, наберите название темы и нажмите Install. Если у Вас появляются ошибки во время установки, попробуйте запустить VS Code от имени администратора или ищите информацию в интернете.
